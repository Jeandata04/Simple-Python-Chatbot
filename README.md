# Python Chatbot

A simple rule-based chatbot built using Python.  
This project was created to practice Python programming, project structure, and version control using Git and GitHub.

## Description

The chatbot responds to user input based on predefined keywords stored in a JSON file.  
If no keyword matches, it returns a default response.

This project demonstrates:
- Basic Python scripting
- Working with JSON files
- Conditional logic and loops
- Simple command-line interaction
- Git workflow (init, commit, push)

## Project Structure

python-chatbot/

## How to Run

1. Clone the repository:
   git clone https://github.com/yourusername/python-chatbot.git

2. Navigate into the folder:
   cd python-chatbot

3. Run the chatbot:
   python chatbot.py

4. Type "quit" to exit.
